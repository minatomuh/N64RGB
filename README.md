# N64RGB
A custom-made DAC for N64 that are not RGB-capable (MAV-NUS / AVDC-NUS / DENC-NUS)

A non-CPLD design that uses discrete components. Suitable for JLCPCB direct order. BOM included.

Have not tested yet.
